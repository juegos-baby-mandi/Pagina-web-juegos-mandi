📌 Condiciones importantes para Juegos Inflables

  • No se permite chaya, confeti, papel picado ni ningún elemento que dañe o ensucie los juegos.
  • La instalación se realiza antes del inicio del evento. Se avisará el horario el día anterior.
  • El cliente debe contar con electricidad en el lugar para el funcionamiento del juego.
  • Una vez funcionando el juego, jamás debe apagarse ni moverse de lugar.
  • Es responsabilidad del cliente medir el espacio donde irá el juego. Considerar el alto (por cables o árboles).
  • Si el juego irá dentro de un recinto, la puerta debe medir al menos 1.20 metros de ancho.
  • No se suben escaleras altas con juegos grandes (pesan sobre 170 kg). Si hay escaleras, debe avisarse antes para evaluar qué juegos se pueden llevar.
